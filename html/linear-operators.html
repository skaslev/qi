Linear operators and data types

Linear Operators

\[ H : \prod_{a,b \in \mathbb{R^n}} \prod_{\alpha,\beta \in \mathbb{R}} H(\alpha a + \beta b) = \alpha H a + \beta H b \] \[ H(\alpha a + \beta b) = \alpha H a + \beta H b \] \[ \| H - \lambda I \| = 0 \] \[ H v_n = \lambda_n v_n \]

List ADT

data [a] = [] | a:[a]

data List a = Nil | Cons a (List a)
\[ L(a) = 1 + a L(a) \] \[ L(a) = \frac{1}{1 - a} \] \[ L(a) - a L(a) - 1 = 0 \] \[ [a^N] L(a) = 1 \]

Tree ADT

type Forrest a = [Tree a]
data Tree a = Node a [Tree a]

data Tree a = Node a (List (Tree a))
\[ T(a) = a L(T(a)) \] \[ T(a) = \frac{a}{1 - T(a)} \] \[ T(a)^2 - T(a) + a = 0 \] \[ T(a) = a + T(a)^2 \]

Binary Tree ADT (external)

data BinTree a = Leaf a | Branch (BinTree a) (BinTree a)
\[ B(a) = a + B(a)^2 \] \[ B(a)^2 - B(a) + a = 0 \]

Full Binary Tree ADT

data BinTree a = Leaf a | Branch a (BinTree a) (BinTree a)
\[ B(a) = a + a B(a)^2 \] \[ a B(a)^2 - B(a) + a = 0 \]

Internal Binary Tree ADT

data BinTree a = Leaf | Branch a (BinTree a) (BinTree a)
\[ B(a) = 1 + a B(a)^2 \] \[ a B(a)^2 - B(a) + 1 = 0 \] \[ [a^N] B(a) = \frac{1}{N+1} \binom{2 N}{N} \sim \frac{4^N}{\sqrt{\pi N^3}} \] \[ a \lambda^2 - \lambda + 1 = 0 \] \[ \lambda_{\pm} = \left\{ \begin{array}{l l} 1 & \quad \text{if $a = 0$}\\ \frac{1 \pm \sqrt{1 - 4a}}{2a} & \quad \text{if $a \neq 0 $} \end{array} \right.\]

Primes

primes = sieve [2..]
sieve (x:xs) = x : sieve [y | y <- xs, y `mod` x /= 0]
