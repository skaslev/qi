Linear operators and data types

Linear Operators

\[ H : \prod_{a,b \in \mathbb{R^n}} \prod_{\alpha,\beta \in \mathbb{R}} H(\alpha a + \beta b) = \alpha H a + \beta H b \] \[ H(\alpha a + \beta b) = \alpha H a + \beta H b \] \[ \| H - \lambda I \| = 0 \] \[ H v_n = \lambda v_n \]

List ADT

data List a = Nil | Cons a (List a)
\[ L(a) = 1 + a L(a) \] \[ L(a) = \frac{1}{1 - a} \] \[ L(a) - a L(a) - 1 = 0 \] \[ L(a) = 1 + a L(a) \] \[ [a^N] L(a) = 1 \]

Tree ADT

data Tree a = Node a (List (Tree a))
\[ T(a) = a L(T(a)) \] \[ T(a) = \frac{a}{1 - T(a)} \] \[ T(a)^2 - T(a) + a = 0 \] \[ T(a) = a + T(a)^2 \]

Binary Tree ADT

data BinTree a = Leaf a | Branch (BinTree a) (BinTree a)
\[ B(a) = a + B(a) B(a) \] \[ B(a)^2 - B(a) + a = 0 \] \[ B(a) = a + B(a)^2 \]

Full Binary Tree ADT

data BinTree a = Leaf a | Branch a (BinTree a) (BinTree a)
\[ B(a) = a + a B(a) B(a) \] \[ a B(a)^2 - B(a) + a = 0 \] \[ B(a) = a + a B(a)^2 \]

Internal Binary Tree ADT

data BinTree a = Leaf | Branch a (BinTree a) (BinTree a)
\[ B(a) = 1 + a B(a) B(a) \] \[ a B(a)^2 - B(a) + a = 0 \] \[ B(a) = 1 + a B(a)^2 \] \[ [a^N] B(a) = \frac{1}{N+1} \binom{2 N}{N} \sim \frac{4^N}{\sqrt{\pi N^3}} \]

Primes

primes = sieve [2..]
sieve (x:xs) = x : sieve [y | y <- xs, y `mod` x /= 0]
